NAME
       chat-client - test package
SYNOPSIS
       chat-client [<host> <port>] or [ OPTION ]
DESCRIPTION
       simple  utility to start your own server with simple command line chat
       which could be coonected to through telnet client.
OPTIONS
       --help,
             Output a usage message and exit.
       change-username,
             Change username in config.json
       remove,
             Remove chat-client package from your computer.
       update,
             Update chat-client to latest version.
       -v,
             Output version info.
USAGE
       - Enter your username if you're starting chat-client first. Then enter
       the name of the room you want connect to.
       - Write your message and press <ENTER> to send it. Only members of
       current room will see it.
       - To leave chat type "exit".
